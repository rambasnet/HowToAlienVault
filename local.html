OSSIM (Open Source Security Information Management) is an open source project by Alienvault which provides the SIEM (Security information and event management) functionality.

It provides event collection, normalization, and correlation features for security professionals.

OSSIM is a unified platform which is providing the essential security capabilities. Many proven open source security software's are built into the OSSIM platform. It continues to be the fastest way to make the first steps towards unified security visibility.

OSSIM platform support following open source software's/plugins:

Apache, IIS, Syslog, Ossec, Snare, Snort, OpenVAS, Nessus, Nagios, Ntop, Nmap
