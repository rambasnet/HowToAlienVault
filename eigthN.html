USM Features Not Included in OSSIM

For this project we have been using the OSSIM version of Alienvault, which is the free version. Alienvault also has a commercial USM version that is $5050, but does have some differences that we wanted to mention.

Feature OSSIM USM
Threat Intelligence NO YES
Open Threat Exchange YES YES
Reports Basic 200+ Custumizable Reports
# Server Deployment One Server Multiple Servers
Documentation Through Alienvault No Documentation Full Documentation

A big difference we noticed in using the OSSIM was the collection of Threat Intellegence

When the OSSIM notices a threat it is not very clear exactly what threat you are detecting/

The OSSIM will basically just sense the high amount of traffic going through a specific port

While the USM Threat Intelligence will interpret the attacks and present them in the Threat Intelligence chart
