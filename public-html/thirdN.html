Running the install ISO

We use eth0 for the management and the rest of the network is connected to eth1. Configuration for eth0 is shown below.

For IP address enter the address you want to set for the web interface for the ACM it will be 172.16.2.***, change the last three digits to an unused port. For Netmask and Gateway enter the routes you obtained in the last step

After network setting, the next window prompts user for the password for root which can access the CLI of OSSIM server

PASSWORD SHOULD BE STRONG

Select your time zone in the next step which is important in the logging system

After setting time zone; wizard automatically perform the partition and start installing the base system.

This process can take 15-20 min

Now the vm will load the CLI where you will need to login with the information you set up during the install to turn on the Web Client

Once you are logged it that CLI will have the IP address of the web client at the top of the terminal, go to that page to access the web client
